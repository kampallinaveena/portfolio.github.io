Info of my work

Project information

  • ToolKit: Eclipse, Visual Studio Code
  • Front-end: HTML, CSS, Angular, Typescript
  • Back-end: Java
  • Database: MySQL

This Application is intended to register citizens and let them display ID to process their Aadhar Card application.

Features of the application:

  • 1.Registration & Login.
  • 2.Apply for a new Aadhar Card & Place a request for updating Aadhar details.
  • 3.Admin: Approve Aadhar Application and issue new Aadhar number.
